# Support Guidelines

Thank you for using Novalabs repositories! If you need help, please use the following channels:

## 🔍 Search Existing Issues
Before raising a support request or opening an issue, please search the repository's open and closed issues.

## 💬 Discussions & Questions
- For general questions, ideas, or architectural discussions, please use the **Discussions** tab (if enabled).
- For bugs and feature requests, please open an **Issue**.
