# Contributing to Novalabs

Thank you for your interest in contributing to Novalabs open-source products! We welcome contributions from developers, researchers, and builders.

## 🚀 How to Contribute

1. **Find an Issue**: Browse the issues in our repositories or create a new one to discuss your ideas.
2. **Fork and Clone**: Fork the repository to your account and clone it locally.
3. **Create a Branch**: Use a descriptive branch name (e.g., `feature/awesome-new-tool` or `bugfix/issue-123`).
4. **Implement & Test**: Write clean code and make sure to include test coverage.
5. **Commit cleanly**: Write clear, descriptive commit messages.
6. **Submit a Pull Request**: Open a PR to the default branch (usually `main`).

## 🤝 Code of Conduct

We expect all contributors to adhere to our [Code of Conduct](CODE_OF_CONDUCT.md).
