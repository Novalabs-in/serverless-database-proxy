# Security Policy

## Supported Versions

We actively support the latest version of this repository. Please ensure you are running the latest version before reporting issues.

## Reporting a Vulnerability

We take the security of our repositories seriously. If you find any vulnerability, please report it privately:

1. Send an email to **saitejabandaru.in@gmail.com**.
2. Include details of the vulnerability and steps to reproduce.
3. Do not open public issues regarding security bugs until we have verified and patched them.

Thank you for helping keep our software secure!
